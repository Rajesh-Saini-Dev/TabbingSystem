A simple and interactive Tabbing System built using React. This application allows users to switch between different tabs and view corresponding content without reloading the page.

Features:
*Switch between multiple tabs<br>
*Active tab state management<br>
*Built using React functional components<br>
*Uses React Hooks (useState)<br>
*Reusable Tab components<br>
*Fully responsive layout<br>

Tech Stack:<br>
*React/Vite<br>
*JavaScript (ES6+)<br>
*Tailwind CSS<br>
